# 徐翔五维共振 · 涨停板量化筛选系统 v2.0

基于徐翔交易哲学的A股涨停板量化筛选系统，八条铁律 + 五维共振策略体系。

## 系统架构

```
main.py           主入口（支持筛选/回测两种模式）
├── config.py     配置中心（dataclass，集中管理所有阈值）
├── data_fetcher  数据采集层（东方财富/新浪财经 + 磁盘缓存）
├── filters.py    铁律筛选引擎（8条一票否决规则）
├── scorer.py     五维评分引擎（加权评分 + 买卖点位）
├── backtester.py 回测引擎（历史胜率验证）
├── presenter.py  输出格式化
└── utils.py      公共工具函数
```

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 收盘后运行（默认当天）
python3 main.py

# 指定交易日
python3 main.py 20260430

# 详细日志
python3 main.py -v

# 指定输出数量
python3 main.py --top 5
```

## 回测模式

```bash
# 回测指定区间
python3 main.py --backtest 20260101 20260430

# 自定义持仓天数
python3 main.py --backtest 20260101 20260430 --hold-days 5

# 或直接运行回测模块
python3 backtester.py 20260101 20260430 3
```

## 运行测试

```bash
pytest tests/ -v
```

## 八条铁律（一票否决）

| # | 规则 | 说明 |
|---|------|------|
| R1 | 标的范围 | 仅限当日涨停，排除14:50后尾盘偷袭 |
| R2 | 价格与流动性 | 股价≤20元 或 流通市值≤1000亿 |
| R3 | 涨停基因 | 近12月自然涨停≥5次（自适应10%/20%板） |
| R4 | 量能验证 | 涨停日量能放大≥30% |
| R5 | 趋势斜率 | 5日线日均斜率≥1% 或 突破120/250日均线 |
| R6 | 上方阻力 | 上方套牢盘≤20% |
| R7 | 封板质量 | 开盘1小时内封板，炸板≤2次 |
| R8 | 风险排除 | 排除ST/天地板/炸板>8% |

## 五维评分

| 维度 | 权重 | 说明 |
|------|------|------|
| D1 涨停板溢价 | 25% | 连板数 + 封板速度 |
| D2 主力资金共振 | 25% | 换手率 + 成交额 + 资金流 |
| D3 全网消息催化 | 15% | 板块热度 + 连板效应 |
| D4 龙头地位确认 | 20% | 连板高度 + 涨停基因密度 |
| D5 量价健康度 | 15% | 量比 + 相对位置 + 均线排列 |

## v2.0 更新

- **回测系统**：新增 `backtester.py`，支持历史胜率验证
- **配置中心化**：所有阈值移至 `config.py` 的 dataclass，便于调优
- **自适应涨停判定**：自动识别10%/20%涨跌幅限制
- **个股资金流**：实现了 `get_individual_fund_flow`（之前为空壳）
- **日志系统**：替换裸 print，支持 `-v` 详细模式
- **类型标注**：全局添加 type hints
- **单元测试**：覆盖核心模块
- **代码去重**：提取 `utils.py` 消除重复代码
- **封板质量增强**：新增炸板次数检查

## 配置调优

编辑 `config.py` 中的 dataclass 即可调整所有参数：

```python
# 示例：放宽价格限制
CFG.iron.max_price = 30.0

# 示例：调整权重
CFG.weights.capital_resonance = 0.30
```
