# 徐翔五维共振 · 涨停板量化筛选系统

基于徐翔交易哲学的A股涨停板量化筛选系统，五维共振策略体系。

## 策略核心

**八条铁律（一票否决）：**
1. 仅限当日涨停股，排除尾盘偷袭（14:50后）
2. 股价≤20元 或 流通市值≤1000亿 + 成交额前200
3. 近12月自然涨停≥5次（涨停基因）
4. 涨停日量能放大≥30%
5. 5日线斜率≥45° 或 涨停突破120/250日均线
6. 上方套牢盘≤20%
7. 开盘1小时内封板，分时直线拉升/三段式
8. 排除近3天天地板/炸板>8%/ST/暴雷

**五维评分：**
- D1 涨停板溢价（25%）
- D2 主力资金共振（25%）
- D3 全网消息催化（15%）
- D4 龙头地位确认（20%）
- D5 量价健康度（15%）

## 使用

```bash
# 安装依赖
pip install akshare pandas tabulate

# 指定日期运行
python3 main.py 20260430

# 默认今天（收盘后运行）
python3 main.py
```

## 数据源

- 涨停池：东方财富（AkShare）
- 历史K线：新浪财经（AkShare）
- 自动磁盘缓存，二次运行秒出
